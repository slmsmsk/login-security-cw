== Description ==

WordPress login page by adding an extra parameter to get rid of the typical input addresses are protected from brute force attacks ..

== Installation ==

1. Upload contents of the directory to `/wp-content/plugins/`  (or use the automatic installer)
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Forgot parameter. How do I log in?

You can use admin email to get login link. (e.g.  http://www.wordpress.org/wp-login.php?email=admin@wordpress.org )
